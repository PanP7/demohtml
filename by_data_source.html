企业中最重要的资产： 数据

从简单的文本文件到TB级的大数据系统, Halo BI帮助你探索所有的数据资源，将它们转化为你决策信息的一部分.


Halo业务分析和情报平台访问下面的数据格式:

  • Relational databases关系数据库
  • Open Source databases开源数据库
  • Flat databases XML
  • Data warehouses数据仓库
  • Spreadsheet files电子表格文件
  • Text files文本文件
  • Operational (ERP, CRM, SCM) and Web application databases
  • 操作型数据库(ERP、CRM、SCM)和Web应用程序数据库
  • Semi-structured & Unstructured big data sources 半结构化和非结构化大数据来源
